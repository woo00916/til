# amam 요구사항 정의

## 회원가입
### 폼 만들기 (~10/6)
* [x] pw / pw confirm / email 입력란이 있을 것
* [ ] 폼 송신 전 validation check가 이루어질 것 
  * [x] email 포맷확인이 이루어질 것 
  * [ ] pw가 pw confirm과 일치하는지 체크할 것 
  * [ ] email, pw, pw confirm은 필수 입력란
* [ ] 폼 송신 시 입력한 데이터가 전부 전송될 것 
### 백엔드
* [ ] 전송받은 내용을 db에 저장할 것
